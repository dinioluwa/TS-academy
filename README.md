# TS-academy
This project performs an Exploratory Data Analysis (EDA) on a car sales dataset to uncover insights through data cleaning, visualization, and statistical analysis about sales patterns, pricing trends, and factors influencing car purchases. Using Python data analysis libraries such as Pandas, NumPy, Matplotlib, and Seaborn, the notebook explores data distributions and relationships between variables.

Objectives

Understand the distribution of car prices, brands, and models
Analyze the impact of market value, transmission, and model on pricing
Identify top-performing brands and regional sales trends
Detect and handle missing values and outliers

Tools & Libraries

Python
Pandas – data manipulation
Matplotlib & Seaborn – data visualization
NumPy – numerical operations
Jupyter Notebook – interactive analysis

Key Insights

Price distribution across different car brands and segments.

Price distribution and yearly trends across different car brands and colors
